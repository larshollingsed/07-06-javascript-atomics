In these exercises, we use prompt() to collect information, but most websites do that ask for information do not use prompt() -- how do they collect information (what HTML elements do they use)? What might be an advantage to doing it that way instead of with prompt()? Are there any disadvantages? Why might you use prompt()?

I assume most websites use a form to collect information from the user directly. With a form you can collect multiple pieces of information (in various formats e.g. typeable fields and a set of checkboxes). The advantage to this is that you can get all of the information from the user in one fell swoop. A disadvantage to using a form may be that the user can accidentally skip parts of the form without noticing, whereas a pop-up from prompt() needs to be dealt with before the rest of the page can be interacted with. I can imagine a situation where using prompt() right as the user loads the page to generate specific parts of that page is necessary.
